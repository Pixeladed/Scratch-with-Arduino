How To:

Program ad control a Piezo buzzer:

After installation of Scratch 4 Arduino (referred to as S4A), students can begin to program some of their own robots using this program. In order to program a Piezo buzzer students must first be provided with these equipment:

  1. An Arduino motherboard
  2. A Piezo buzzer

After making sure S4A is functioning correctly with Arduino, the students can start programming the Arduino using S4A by following these steps:

Code the Program

In order to tell the Arduino to do something we must first pick a starting key to press in order to initiate the command to the Arduino. We can do this by going to the menu in the top left of the S4A interface and selecting the box labelled control. On the left there should be many options to choose from, scan the menu and find the when space key pressed block and drag it to the blank space in the middle.

Then grab a forever loop from the side bar and place it in the space in the middle. After, grab one digital 10 on and one digital 10 off from the motion menu and place them inside the forever loop. Change the numbers from 10 to 11 inside both digital 10 on/off blocks. Return to the control menu and grab 2 wait 1 secs blocks and place each under both digital 11 on and digital 11 off blocks, within the forever loop.

Once that has been completed, select the variables box in the top left menu and create a variable. Name this variable “freq”, for frequency, and drag a set freq to 0 block from the side bar. Place it between the when space key pressed and forever loop blocks and change the number from 0 to 0.5 . Grab 2 freq variable blocks and replace the 1 in both wait 1 secs blocks with them. Setting up Arduino

Connect the 3 pronged Piezo buzzer into ports 9, 10 and 11 into the Arduino motherboard and press space to the start the beeping sequence. Changing the value of x in set freq to x will alter the rate that the Piezo buzzer will beep. Higher will decrease the rate, lower will increase the rate.
